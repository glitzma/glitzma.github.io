# 思维导图

# jsx基本使用

与vue的模板不同的是变量和表达式写在{}中，事件的绑定是onXxxx的方式。

一、 条件判断可以是：

  1. if else
  2. 三元表达式
  3. 逻辑运算符 && ||

二、 渲染列表

用map遍历。 记得加key。 foreach遍历没有返回值。map有。

三、 事件

  • bind this。可以写以constructor 可以写在dom中。 如果是箭头函数定义方法不需要bind。写在dom上每次点击都会bind
  • event参数 与vue不同的是。vue的event是原生的。react的event是合成的叫syntheticEvent。event.nativeEvent是原生事件对象。react中所有事件都挂载到document上。syntheticEvent模拟出来dom事件所有能力。

四、 表单

  • 受控组件（类似于vue双向数据绑定）

五、组件的使用

props传递数据、函数。props类型检查，需要安装'prop-types'插件，进行类型检查。

六、状态（数据）提升

数据要放到高级别的组件中管理

七、 setState

  • 不可变值
  • 可能异步更新
  • 可能会被合并
  1. state要在构造函数中定义
  2. 不要直接修改state，要使用不可变值
  3. 在setTimeOut中、在componentDidMount自定义的dom事件中。setState是同步的，其它是异步的
  4. state异步更新的话。如果传入的是对象则会被合并（类似于object.assign)，如果传入是函数，则不会合并更新

# react生命周期

一、 常用的生命周期

挂载时：constructor、render、componentDidiMount 更新时： render、componentDidUpdate 卸载时： componentWillUnmount

二、 不常用的生命周期

getDerivedStateFromProps、shouldComponentUpdate、render、getSnapshotBeforeUpdate
