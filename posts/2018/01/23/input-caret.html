# android手机可以完美隐藏，ios手机会出现光标

尝试方法（input 移动到视线外，qq浏览器会卡顿）

1. 把input 放在 div 后面
   div 设置  touch-action: none
2. input定位到可视范围外
3. visibility 设置成  hidden
4. color: transparent;
5. text-indent: -99999999px; margin-left: -100%; width: 200%;
6. font-size:0的话， 到了android 上就完蛋了。但ios上可以用

解决方案

我们是vue项目
在mounted中加入手机机型判断
如果是android就用正常的隐藏就好
如果是ios则input的font-size:0
